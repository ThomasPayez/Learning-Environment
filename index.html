Fonctionnement de Git

## *Les différentes commandes avec Git* ### *git init :* Initialise un nouveau dépôt Git. ### *git config :* Raccourci de “configurer,” ceci est tout particulièrement utile quand vous paramétrez Git pour la première fois. ### *git help :* Permet d'afficher les 21 commandes les plus utilisées. ### *git add :* Ceci n’ajoute pas de nouveaux fichiers dans votre repository. Au lieu de cela, ceci porte de nouveaux fichiers à l’attention de Git. Après avoir ajouté des fichiers, ils sont inclus dans les « instantanés » du dépôt Git.
