Fonctionnement de Git

Les differentes commandes avec Git

git init : Initialise un nouveau depot Git.

git config : Raccourci de “configurer,” ceci est tout particulierement utile quand vous parametrez Git pour la premiere fois.

git help : Permet d'afficher les 21 commandes les plus utilisees.

git add : Ceci n’ajoute pas de nouveaux fichiers dans votre repository. Au lieu de cela, ceci porte de nouveaux fichiers a l’attention de Git. Apres avoir ajoute des fichiers, ils sont inclus dans les « instantanes » du depot Git.
